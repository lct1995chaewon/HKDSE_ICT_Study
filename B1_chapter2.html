第二課 電腦硬件

2.1 主機與核心部件

主機板 (Motherboard)

  • 載有插槽、連接埠與匯流排，連接 CPU、RAM、儲存與周邊。
  • USB、PCIe、SATA 等介面統一在主機板上。

處理器 (CPU / GPU)

  • CPU：擅長繁複指令、通用處理。
  • GPU：核心數多，擅長並行簡單運算，圖像與 AI 訓練常用。
  • iGPU：GPU 整合在 CPU/晶片組內，無需獨立顯示卡。

圖形卡 (顯示卡)

  • 包含 GPU、專用顯示 RAM、數碼模擬轉換器。
  • 專為圖像輸出設計，可連接顯示器/投影機。
  • 🚨顯示卡不等於GPU

主記憶體與輔助儲存

  • RAM（易失性）暫存處理數據；ROM（非易失性）儲 BIOS。
  • 硬碟 / SSD / 光碟等屬輔助儲存，電源關閉後仍保留數據。

電源供應器 (PSU)

  • 將 AC 轉為各部件需要的 DC 電壓。
  • 功率需符合整體硬件需求。

匯流排 (Bus)

  • 系統匯流排：含位址、數據、控制匯流排。
  • USB：連接外部設備，1.0/2.0/3.0 可由插頭顏色分辨。

互動：選擇部件查看重點

2.2 中央處理器 (CPU)

主要部件

  • CU（控制部件）：協調指令、管理資料流。
  • ALU（算術邏輯單元）：執行 + − × ÷ 及邏輯比較、AND/OR/NOT。
  • 寄存器：CPU 內高速暫存區，如 PC、MAR、MDR、CIR、累加器。
  • 快取記憶體：比 RAM 更快，預先儲常用指令/數據。

機器周期（馮紐曼架構）

  1. 提取 (Fetch)：PC 指向下一指令位址 → 送往 MAR/MDR。
  2. 譯碼 (Decode)：CU 解析指令。
  3. 執行 (Execute)：ALU 或其他單元進行操作。
  4. 儲存 (Store)：結果寫回寄存器或記憶體。

規格對表現的影響

  • 時鐘頻率（GHz）：每秒時計畫週期數，越高越快亦較耗電。
  • 核心數：多核心提升多工作業能力。
  • 字長（32/64 bit）：每次處理位元數，影響定址能力與速度。

2.3 主記憶體與輔助儲存

RAM

  • 易失性；電腦關機後數據消失。
  • 存取速度遠高於輔助儲存設備。
  • 容量與速度直接左右 CPU 表現。
  • 版本：DDR3/DDR4/DDR5，插槽不兼容。

ROM

  • 非易失性，只讀或極少改寫。
  • 儲存 BIOS / 韌體，負責啟動程序。

輔助儲存分類

  • 磁儲存：硬碟、磁帶。
  • 固態儲存：SSD、USB、記憶卡（使用快閃記憶體）。
  • 光儲存：CD/DVD/Blu-ray。

存取方式

  • 順序存取：磁帶，只能依次讀寫。
  • 直接存取：硬碟、SSD、USB，可隨機定位。

其他技術

  • RAID：多儲存組合提升速度與冗餘。
  • NAS：LAN 共享儲存，常結合 RAID。
  • 雲端儲存：ISP 提供，需網絡連線。

記憶體階層概念

階層 例子 速度 容量 用途
最快 寄存器 極高 極小 暫存即時指令 / 算術結果
高 CPU 快取 高 小 預存常用數據
中 RAM 中 中 暫存執行中程式資料
較低 SSD / HDD 較低 大 長期儲存
最低 磁帶 / 雲端備份 低 極大 備份、歸檔

2.4 電腦種類與應用

桌上電腦

  • 運算力強、擴充性高。
  • 適合圖像設計、遊戲、伺服器。

手提電腦

  • 便於攜帶，性能中度。
  • 電池供電，整合顯示器與鍵盤。

平板 / 智能電話

  • 重量輕、觸控操作。
  • 運算較低但耗電少，適合流動多媒體。

伺服器 / 超級電腦

  • 伺服器：連線多用戶、資料共享。
  • 超級電腦：極強並行運算，如科學模擬。

2.5 硬件最新發展趨勢

CPU / GPU

  • 更多核心、更高時鐘頻率。
  • AI 加速器、GPU 成為深度學習核心。

儲存

  • SSD 逐漸取代 HDD 作系統碟，速度快、耐震、耗電低。
  • 快閃記憶體密度持續提升。

行動裝置

  • 螢幕解像度、刷新率提高。
  • 運算能力增強、AI 相機功能普及。

物聯網 / 感應器

  • 更多裝置內建感測與無線模組。
  • 感測數據結合雲端與 AI 分析。
