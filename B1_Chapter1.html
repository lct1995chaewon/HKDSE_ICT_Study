第一課｜輸入與輸出設備 溫習簡報

1.1 輸入－處理－輸出 (IPO) 循環

  1. 輸入：藉輸入設備取得指令或感測數據（鍵盤、感應器等）。
  2. 處理：程式執行運算、更新或提取儲存的數據。
  3. 輸出：以顯示器、打印機、揚聲器等呈現有意義的資訊。
感測器日常例子：
  • 自動門：紅外線 / 超聲波感應器偵測有人靠近。
  • 自動沖水：距離或紅外線感應器偵測離開。
  • 冷氣恆溫：温度與濕度感應器回饋予控制系統。

1.2 輸入設備分類

鍵盤類

  • 全尺寸鍵盤：>100鍵，輸入 ASCII / Unicode 代碼。
  • 數字鍵盤：銀行、POS 輸入 PIN / 金額。

定點設備

  • 動作追蹤：滑鼠、軌跡球。
  • 位置追蹤：觸控板、觸控屏。

音訊與樂器

  • 麥克風：把聲波轉成數碼音訊，可做語音識別。
  • MIDI 鍵盤：輸入音符資料，不受環境噪聲。

圖像與視像

  • 掃描器（平板、3D）：將紙本轉成圖像；配 OCR 得文字。
  • 數碼相機 / 錄影機 / 網絡攝影機：拍照、錄影或串流。

自動輸入

  • 條碼、二維碼掃描器：迅速輸入產品 ID / 字串。
  • 光學標記閱讀器、磁帶卡與智能卡閱讀器。
  • 感應器：光、溫濕度、超聲波、陀螺儀等。
  • 生物識別：指紋、臉部、虹膜、聲紋。

互動：選擇輸入類型查看要點

1.3 輸出設備

顯示器

  • 關鍵規格：尺寸、解像度、像素密度(ppi)、色深、長寬比、更新率。
  • 常見介面：VGA、DVI、HDMI、DisplayPort。

投影機

  • 適合多人或可變尺寸畫面。
  • 需考慮環境光、便攜性。

打印機

  • 感熱式：快速、低成本、收據用。
  • 噴墨：色彩細膩，墨盒易堵。
  • 激光：速度快、遇水不暈、每頁成本低。
  • 繪圖儀：高精度向量圖。
  • 3D 打印：把 3D 模型實體化。

聲音輸出

  • 揚聲器：覆蓋廣、易受噪音。
  • 頭戴式/入耳式耳機：個人化、可整合麥克風。

1.4 電腦應用實例

商業用途

  • 視像會議：遠端協作。
  • POS / 自助結賬：條碼掃描器 + 智能卡閱讀器 + 感熱/激光打印機。
  • 電子支付、電子收據。

教育與公共服務

  • 網上課程、視像授課。
  • 圖書館目錄終端、自助借書機。
  • 博物館資訊站提供互動解說。
